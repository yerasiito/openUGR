\doxysubsection{Detailed Description}
\subsubsection*{template$<$class T$>$\newline
class Pila$<$ T $>$}
T.\+D.\+A. \mbox{\hyperlink{classPila}{Pila}}. 

Una instancia {\itshape v} del tipo de datos abstracto \mbox{\hyperlink{classPila}{Pila}} sobre el tipo {\ttfamily T} es una lista de elementos del mismo con un funcionamiento {\itshape LIFO} (Last In, First Out). En una pila, las operaciones de inserción y borrado de elementos tienen lugar en uno de los extremos denominado {\itshape Tope}. Una pila de longitud {\itshape n} la denotamos


\begin{DoxyItemize}
\item \mbox{[}a1,a2,a3,..,an\texorpdfstring{$>$}{>}
\end{DoxyItemize}

donde {\itshape ai} es el elemento de la posición i-\/ésima.

En esta pila, tendremos acceso únicamente al elemento del {\itshape Tope}, es decir, a {\itshape an}. El borrado o consulta de un elemento será sobre {\itshape an}, y la inserción de un nuevo elemento se hará sobre éste. Quedando el elemento insertado como {\itshape Tope} de la pila.

Si {\itshape n=0} diremos que la pila esté vacía.

El espacio requerido para el almacenamiento es O(n). Donde n es el número de elementos de la \mbox{\hyperlink{classPila}{Pila}}.
