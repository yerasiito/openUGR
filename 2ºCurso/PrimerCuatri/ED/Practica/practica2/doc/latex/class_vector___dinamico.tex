Una instancia {\itshape v} del tipo de datos abstracto {\ttfamily \mbox{\hyperlink{class_vector___dinamico}{Vector\+\_\+\+Dinamico}}} sobre el tipo {\ttfamily float} es un array 1-\/dimensional de un determinado tamaño {\itshape n}, que puede crecer y decrecer a petición del usuario. Lo podemos representar como

\{v\mbox{[}0\mbox{]},v\mbox{[}1\mbox{]},...,v\mbox{[}n-\/1\mbox{]}\}

donde v\mbox{[}i\mbox{]} es el valor almacenado en la posición i del vector

La eficiencia en espacio es {\itshape O(n)}.
